🎮 6 шагов запуска

🔹 ШАГ 1. СОЗДАНИЕ ИГРЫ

Учитель: нажимает «СОЗДАТЬ» → появляется код игры (например, 12345678). В панели учителя статус «👥 УЧЕНИКИ (0)» — ожидание подключения.

🔹 ШАГ 2. ПОДКЛЮЧЕНИЕ УЧЕНИКА

Ученик: вводит имя (например, Анна) и код игры, нажимает «ПОДКЛЮЧИТЬСЯ».
Учитель видит: имя ученика и «0 очков», счётчик «👥 УЧЕНИКИ (1)».

🔹 ШАГ 3. ЗАПУСК ВОПРОСА

Учитель: нажимает «ВОПРОС».
Ученик видит: вопрос (например, «Что такое фишинг?») и 4 варианта ответа.

🔹 ШАГ 4. ОТВЕТ УЧЕНИКА

Ученик: выбирает вариант (например, «Кража данных»).
Учитель видит: статистику: кто ответил, какой вариант, статус (✅ — ответил, ⏳ — ожидание).

🔹 ШАГ 5. ПОКАЗ ПРАВИЛЬНОГО ОТВЕТА

Учитель: нажимает «ОТВЕТ».
Ученик видит: правильный ответ (+1 очко) и объяснение. Например: «Фишинг — мошенничество для кражи данных».

🔹 ШАГ 6. ТАБЛИЦА ЛИДЕРОВ

После всех вопросов — полная статистика: список учеников с очками, таблица ответов по каждому вопросу, кнопки «СКАЧАТЬ» и «СБРОСИТЬ».

⚠️ Пример для темы «кибербезопасность». Принцип одинаков для всех викторин.
