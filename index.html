О платформе

Данная инструкция познакомит вас с работой образовательной платформы "ИльяКласс" и системой сбора предложений для создания новых викторин.

🎯 Цель Освоение навигации между викторинами и взаимодействие с платформой
👥 Аудитория Педагоги, организаторы мастер-классов, участники образовательного процесса
⏱️ Время 15-20 минут на изучение материалов

Используйте кнопки внизу для перехода между слайдами.

📌 Основная платформа

Название
ИльяКласс (демо-версия)
Ссылка
https://ilyasigma111.github.io/ilyaklass-demo
Описание
Главная страница, на которой размещены все викторины. С помощью этого интерфейса осуществляется переход между заданиями и навигация по платформе.

🔹 Особенность: при нажатии на кнопки отображаются различные информационные факты

💡 Система сбора предложений

Название
ПРЕДЛОЖКА
Ссылка
https://ilyasigma111.github.io/PredlozhkaSite
Назначение
Специализированная форма для сбора идей от участников. Сюда можно направлять предложения по темам новых викторин, а также запросы на добавление функционала.

Все отправленные предложения отображаются в общем списке в реальном времени.

⚙️ Техническая информация

Структура Каждая викторина размещена в отдельном репозитории на GitHub
Платформа GitHub Pages — статический хостинг для демонстрации проектов
Доступ Открытый, не требует регистрации или авторизации

Репозиторий проекта: github.com/ilyasigma111

📋 Практическая часть

  • 1 Открыть основную платформу по ссылке из слайда 2
  • 2 Изучить навигацию и ознакомиться с интерфейсом
  • 3 Перейти в форму сбора предложений (слайд 3)
  • 4 Сформулировать и отправить тему для будущей викторины

После выполнения заданий можно вернуться к любому слайду для повторного изучения материалов.
