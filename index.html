🎮 6 шагов запуска игры

🔹 ШАГ 1. СОЗДАНИЕ ИГРЫ

Действие учителя: Нажимает кнопку «СОЗДАТЬ». На экране появляется уникальный код игры (например, 12345678). В панели учителя отображается статус «👥 УЧЕНИКИ (0)» — ожидание подключения участников.

🔹 ШАГ 2. ПОДКЛЮЧЕНИЕ УЧЕНИКА

Действие ученика: На своём устройстве (телефон/планшет) вводит имя (например, Анна) и код игры, затем нажимает «ПОДКЛЮЧИТЬСЯ».
Что видит учитель: В панели появляется запись «Анна — 0 очков», счётчик меняется на «👥 УЧЕНИКИ (1)».

🔹 ШАГ 3. ЗАПУСК ВОПРОСА

Действие учителя: Нажимает кнопку «ВОПРОС».
Что видит ученик: На экране отображается вопрос (например, «Что такое фишинг?») и 4 варианта ответа.

🔹 ШАГ 4. ОТВЕТ УЧЕНИКА

Действие ученика: Выбирает один из вариантов (например, «Кража данных»).
Что видит учитель: В панели появляется статистика: какие ученики ответили, какой вариант выбрали, статус (✅ — ответил, ⏳ — ожидание).

🔹 ШАГ 5. ПОКАЗ ПРАВИЛЬНОГО ОТВЕТА

Действие учителя: Нажимает «ОТВЕТ».
Что видит ученик: Правильный ответ (например, «Кража данных») с пометкой +1 очко, а также краткое объяснение. Пример: «Фишинг — вид интернет-мошенничества для кражи данных».

🔹 ШАГ 6. ТАБЛИЦА ЛИДЕРОВ

После завершения всех вопросов: Отображается полная статистика:
  • Список учеников с итоговыми очками
  • Таблица ответов по каждому вопросу (сколько ответили, сколько правильно)
  • Кнопки «СКАЧАТЬ» (экспорт данных) и «СБРОСИТЬ» (новая игра)

⚠️ Пример приведён для темы «кибербезопасность», но алгоритм работы одинаков для всех викторин.
